# Customer-Purchase-Behaviour-Analysis
Customer Purchase Behaviour Analysis is a SQL-based data analysis project focused on understanding how customers interact with products and contribute to revenue. The project analyzes purchasing patterns, top customers, product performance, country-wise sales, and monthly trends using relational databases and SQL queries.
# Customer Purchase Behaviour Analysis Using SQL

## Project Overview
This project analyzes customer purchase behavior using SQL to uncover insights such as top customers, popular products, and sales trends.

## Tools Used
- SQL (SQLite/MySQL)
- DB Browser for SQLite

## Database Structure
- Customers
- Products
- Orders

## Key Analysis
- Total revenue
- Top customers by spending
- Product popularity
- Country-wise sales
- Monthly sales trend

## Insights
- USA customers contribute the highest revenue
- Laptops generate the most sales
- Repeat customers drive more revenue
